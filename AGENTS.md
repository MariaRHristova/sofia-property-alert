# Project Agent Instructions

After completing or testing a meaningful project module, making an important architecture decision, resolving a substantial challenge, comparing AI tools, or capturing working-system evidence, follow the project-local skill at `skills/update-exam-evidence/SKILL.md` and update `docs/exam-journal.md` in the same change.

Do not update the journal for trivial formatting-only edits. Never invent prompts, test results, screenshots, or tool usage, and never include secrets.
